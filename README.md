# Financius

A full-stack web app for a couple to manage shared expenses and income.

## Important Project Docs

- `AGENTS.md` (root): source of truth for project phases and requirements.
- Detailed documentation lives in [`/docs`](./docs/README.md).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Database

```bash
npm run db:migrate
npm run db:seed
```

## Testing

```bash
npm test
npm run test:e2e
```
